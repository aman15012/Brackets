Keywords


Reduced Echelon

A matrix is in reduced row echelon form (also called row canonical form) if it satisfies the following conditions: It is in row echelon form. Every leading coefficient is 1 and is the only nonzero entry in its column.


Determinant

Determinant is a function which as an input accepts matrix and out put is a real or a complex number that is called the determinant of the input matrix.


Diagonal Matrix

A matrix in which all non-diagonal elemints are zero.


Transpose Matrix

In the transpose matrix, the element at row r and column c of the original matrix is placed at row c and column r.


Symmetric Matrix

A symmetric matrix is a matrix whose transpose is equal to itself.


Skew-Symmetric Matrix

A skew-symmetric matrix is a matrix whose transpose is equal to the negative itself.


Invertiblity

A matrix is invertible if it has an inverse.
